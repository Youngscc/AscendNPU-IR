# Agent Startup Instructions

Before working in this repository, read the task memory index:

```text
.agent/README.md
```

The `.agent/` directory contains agent-facing context for the current PlanMemory
overflow memory-estimation task. It is intentionally separate from `docs/` and
should be updated when new facts, experiments, or implementation decisions are
made.

Do not record private user IR contents in `.agent/` unless the user explicitly
allows it.
