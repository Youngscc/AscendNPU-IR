快速开始
========

本节帮助您完成 AscendNPU IR 的构建并运行第一个示例。

- :doc:`构建安装 <installing_guide_zh>` — 环境与工具链要求、源码/二进制构建步骤、CANN 安装与测试运行。
- :doc:`示例 <examples_zh>` — 使用 ``bishengir-compile`` 编译示例 IR，并在设备端通过 CANN 运行时执行。

前置条件：支持的编译器（如 Clang ≥ 10）、CMake ≥ 3.28、Ninja；端到端运行需 CANN 与昇腾硬件，详见安装指南。
