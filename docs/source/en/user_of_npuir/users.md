# AscendNPU IR users

This project supports the integration of multiple operator programming frameworks with the Ascend backend and delivers Ascend-oriented compilation optimization capabilities. The following are examples of languages and frameworks that have been integrated with or use AscendNPU IR.

## Language ecosystem

| DSL | Description |
| --- | --- |
| [Triton-Ascend](https://gitcode.com/Ascend/triton-ascend) | Empowers Triton developers with rapid Ascend operator development and ecosystem migration capabilities |
| [TileLang-Ascend (branch npuir)](https://github.com/tile-ai/tilelang-ascend/tree/npuir) | A tile-level programming framework that streamlines high-performance operator development, balancing development efficiency and low-level hardware optimization capabilities |
| [DLCompiler](https://github.com/DeepLink-org/DLCompiler) | A Triton-extended deep learning compiler featuring cross-architecture DSL extension and intelligent automatic optimization capabilities |
| [FlagTree](https://github.com/flagos-ai/flagtree) | An open-source AI compiler based on Triton, providing unified compilation capabilities for multiple backends |
